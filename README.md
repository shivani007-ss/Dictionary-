# Dictionary-
This is a simple Dictionary web application built as a beginner project. It allows users to search for the meanings of words quickly using a clean and user-friendly interface that translate simple words into many different languages.

## 🚀 Live Demo
👉 https://lasting-unrealistic-debugger--shivanispy007.replit.app

## ✨ Features
- Search for word meanings  
- Simple and responsive design  
- Easy to use  

## 🛠️ Tech Stack
- HTML  
- CSS  
- JavaScript  

## 🎯 Purpose
This project was created to practice web development skills and understand how to build interactive web applications using basic technologies. It focuses on simplicity, usability, and improving frontend development knowledge.
